Basiske datatyper

Nedenstående er en liste af forskellige datatyper

  • number
  • String
  • Boolean

    • En boolean kan være true eller false

Boolske operatorer
Operator Udtryk Eksempel True or false?
< Mindre end 2 < 5 true
<= mindre end eller lig med 5 <= 5 true
== lig med 3 == 3 true
!= ikke lig med 3!=3 false
> større end 2>1 true
>= større end eller lig med 2>=3 false
=== lig med og samme type

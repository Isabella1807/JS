JavaScript Notes

Data Types

  • Number
    Any number, including numbers with decimals: 4, 8, 1516, 23.42.
  • String
    Any grouping of characters on your keyboard (letters, numbers, spaces, symbols, etc.) surrounded by single quotes: ' ... ' or double quotes " ... ", though we prefer single quotes. Some people like to think of string as a fancy word for text.

      let a = 'Hello' + ' ' + 'world';
      console.log(a);
      logs "Hello world" in the console

  • Boolean
    This data type only has two possible values— either true or false (without quotes). It’s helpful to think of booleans as on and off switches or as the answers to a “yes” or “no” question.
  • Null
    This data type represents the intentional absence of a value, and is represented by the keyword null (without quotes).
  • Undefined
    This data type is denoted by the keyword undefined (without quotes). It also represents the absence of a value though it has a different use than null. undefined means that a given value does not exist.
  • Symbol
    A newer feature to the language, symbols are unique identifiers, useful in more complex coding. No need to worry about these for now.
  • Object
    Collections of related data.

Properties

String properties

  • .length
    every string instance has a property called length that stores the number of characters in that string. You can retrieve property information by appending the string with a period and the property name:

      console.log('my string'.length);
      Will log the amount of characters in the above strong, which is 9 here.

Built-in Objects

Built-in Objects are collections of methods and properties that JavaScript

  • Math
    Math contains a few methods such as:
    • .random()
    • .floor()
    • .ceil()

    console.log(Math.random() * 50);
    will log a random number, most likely with decimals from 0 to 50

    console.log(Math.floor(Math.random() * 50));
    will log a random number without decimals from 0 to 50

    console.log(Math.floor(Math.random() * 10) + 90);
    will log a random number without decimals in a range of 10, from 90 and up.

    console.log(Math.ceil(43.8));
    will log a the closest integer to the number in parenthesis.

Variables

a variable is a container for a value. Below are the keywords to used to create variables:

  • var
  • let

    • variables assigned using let can have their value be reassigned a different value.

    let fruit = 'apple';
    console.log(fruit); logs apple
    fruit = 'pear';
    console.log(fruit); logs pear

  • const

    • A const variable cannot be reassigned because it is constant.

    If reassigned a new value, you will get a TypeError.
